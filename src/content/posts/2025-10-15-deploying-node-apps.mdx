# Deploying Node.js Apps

This guide covers Dockerizing a Node.js app and deploying to a cloud provider.

- Write a small `Dockerfile`
- Push to container registry and deploy

```dockerfile
FROM node:20
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
CMD ["node","dist/index.js"]
```
